# Matrix
## purpose of the exercice

The purpose is to rotate N-Matrix 


        [ a b c ]            [ d a b ]
    X = [ d e f ]    ->  v = [ g e c ]
        [ g h i ]            [ h i f ]


how to use 

```
(new Rotate(require 'cases/xxx'))
    ->process()
    ->display();
```

for more details look at `Matrix/rotater.php` or if you want a procedural version please refer to `Matrix/matrix.php`
